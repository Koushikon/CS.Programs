### File `global.json` for .Net:
`global.json`: Used for Set Root SDK Version with this file in Sub-Directory we cannnot Run Apps with Other SDK version. `global.json` Contain:
```JSON
{
  "sdk": {
    "version": "5.0.402", // "6.0.100" or "7.0.200"
  }
}
```
